# Politique de confidentialité - GuinéeMarket

*Dernière mise à jour : 2025*

## 1. Collecte des données

GuinéeMarket collecte les informations suivantes :

| Donnée | Usage |
|--------|-------|
| Numéro de téléphone | Authentification, notifications SMS |
| Nom complet | Identification sur la plateforme |
| Adresse email (optionnel) | Communications, récupération de compte |
| Ville / Région | Affichage des annonces par localisation |
| Numéro CNI (marchands) | Vérification KYC |
| Historique de transactions | Suivi des commandes et litiges |

## 2. Utilisation des données

- Mise en relation acheteurs / vendeurs
- Traitement des paiements via **MobilePay Guinée**
- Notifications SMS liées aux commandes
- Amélioration du service

## 3. Partage des données

- **MobilePay Guinée** : numéro de téléphone pour les transactions
- **Passerelle SMS** : numéro pour l'envoi de notifications
- Aucune donnée vendue à des tiers

## 4. Sécurité

- Données chiffrées en transit (HTTPS / TLS)
- Mots de passe hashés avec bcrypt
- Clés API stockées hors du code source
- Accès restreint aux données personnelles

## 5. Conservation

Les données sont conservées pendant la durée d'utilisation du compte, puis 12 mois après suppression.

## 6. Vos droits

Conformément à la réglementation guinéenne, vous pouvez :
- Accéder à vos données
- Rectifier vos informations
- Supprimer votre compte
- Exporter vos données

Contact : **privacy@guineemarket.gn**

## 7. Cookies

Utilisation de cookies techniques uniquement (session, préférences de langue).
